Track Inbound & Outbound Link Clicks Easily

== Description ==

DW Shortlinks is a WordPress plugin helps you create short and clean URLs, both inbound and outbound. It also helps you to know how many clicks on your link. This plugin is suitable for Affiliate Maketters that can create their own reference link without making their readers feel annoying. Or if you want to create download links but do not want others to copy it or share it back on their website, DW Shortlinks is the right plugin for you to do that.

<h4>Major Features</h4>
* Create short and clean URLs, with your own domain
* Redirect links to any location, both inbound and outbound, using 301 redirects
* Track outbound link clicks, with internal reporting for hits per link
* Create multiple link types with your own name

<h4>How it Works?</h4>
DW Shortlinks adds a new custom post type to your Admin menu, where you can create, edit, delete, and manage short links by 301 redirects. What's great about DW Shortlinks is that it tracks click counts and saves them as a custom field. it's really simple & cool and there is no setup involved. It works seamlessly straight out of the box for whichever theme and plugins you are using.

== Installation ==

1. Upload plugin files to your plugins folder or install using WordPress built-in Add New Plugin installer
2. Activate the plugin
3. Navigate to Settings > Permalinks and save them. Yes, just click Save Changes.
4. Go to the Linker Menu under the Pages
5. Create a new Link and publish
6. Now you can track how many times clicked on each link
